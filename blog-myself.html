Do you Know What Thing Really Makes Artist Nowadays?

Programmer - Developer - Artist

Everyone has their own story. A story about their passion, story related to their career, a success story or failure stories. Stories help us find a way to a good future and also give us knowledge.

Firstly, I got my first personal computer at the age of 10. As every child does I thought I could build something great at that time. I started learning with Microsoft paint which was the first software I used ever. I started learning the Microsoft office onwards.

Then I started playing games and using applications provided by Microsoft with Windows XP. I started using Windows Movie Maker, Virtual Dj, etc software for content creation. My interest in being a graphic artist has increased and I started learning adobe photoshop cs2 (2004) at that time which was the best image manipulation tool available.

Playing with adobe photoshop was awesome while completing academics and classical singing courses. I learned classical singing until my 10th std. I completed the Engineering Drawing course of three years along with Elementary and Intermediate Drawing courses. That built my designing skills.

I was eager to invest my skills into a product so I made some research and learned some applications provided by Adobe.and started developing UI/UX designs. I enrolled for a c programming course in the 8th standard, after that I learned c++ on my own. started building logic and problem-solving strategies.started with java which I was mostly interested in.

After that, I tried C / C++ for development which was awesome using Windows GUI and QT. I also started working with an android studio and studying for core java concepts. I realized that only one programming language is not sufficient to build an application it needs to know some tools also. Then I started learning Database Management Systems. I used MySQL, Xampp, and PHP. after that I shifted to NoSQL databases. I started using Google Firebase which was an awesome tool for creating mobile and web applications. I would like to learn about newer technologies and improve my skills.

I call myself a programmer, developer, and artist. I would like to explore the latest technology and We know we are the bright future of this world.
